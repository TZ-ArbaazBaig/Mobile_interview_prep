# Project Overview

**mobile_interview_prep** is a Flutter‑based mobile application designed to help developers practice technical interview questions. It provides a mock interview experience with real‑time drafting, auto‑save, instant evaluation, and a polished dark‑theme UI.

---

## Features

- **Interview Flow** – configurable question categories (Technical, Behavioral, System Design) with progress indicator.
- **Answer Drafting** – multi‑line input, auto‑saved every few seconds.
- **Evaluation** – immediate feedback and scoring after each answer.
- **Results Dashboard** – summary of overall performance and per‑question details.
- **Theming** – custom violet accent palette for a premium look.

---

## Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/TZ-ArbaazBaig/Mobile_interview_prep.git
   cd Mobile_interview_prep
   ```
2. **Install dependencies**
   ```bash
   flutter pub get
   ```
3. **Run the app**
   ```bash
   flutter run
   ```
